Description:
Convert number to reversed array of digits
Given a random non-negative number, you have to return the digits of this number within an array in reverse order.

Sample Tests:

Input:35231
Output:[1,3,2,5,3]

Input:0
Output:[0]

Input:23582357
Output:[7,5,3,2,8,5,3,2]

Input:45762893920
Output:[0,2,9,3,9,8,2,6,7,5,4]

Time Complexity:
->In ConvertNumberToReversedArrayOfDigits.py
all methods have a time complexity of O(log n), where n is the number being reversed.
